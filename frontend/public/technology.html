How Plainly works

How Plainly explains documents
in plain English.

Plainly helps organisations make important forms, letters and agreements easier to understand — using approved document sets, plain-English explanations, and built-in reading support.

Approved documents

Built around the documents you choose.

Plainly is configured around selected forms, letters and agreements. This keeps the experience focused, relevant, and safer than trying to support every possible document type.

Focused experience

Each organisation chooses the document types Plainly supports. Users work with approved content — not an open upload tool.

Safer and more relevant

Controlled document sets mean Plainly can apply the right rules, the right vocabulary, and the right context for each document type.

Sector-specific rules

Plainly includes rule sets for government, health, legal, tax, insurance, property, employment, and health and safety documents — each mapped to the relevant NZ legislation.

Quality checking built in

Every explanation can be checked against the original document for accuracy, completeness, and plain-language quality.

Reading support

Plain-English explanations with reading support built in.

Plainly explains supported documents in plain English, with tools that help people read and understand the content.

Side-by-side view

The original document and the plain-English explanation appear together on screen. No switching tabs, no losing context.

Text to speech

Every explanation can be read aloud using the browser's built-in speech engine. Click any word to start playback. Adjust speed. Follow along with word highlighting. No API cost.

Privacy and trust

Private by design.

Plainly processes documents in real time. No documents are stored. No user accounts are required. No personal data is retained.

No document storage

Documents are processed and the result is returned. The original document is not saved, logged, or retained.

No user accounts

People using Plainly do not need to create an account or provide personal information.

AI with guardrails

Plainly uses Anthropic's Claude AI. Every explanation follows strict accuracy rules — no advice, no interpretation, no guessing. If something is unclear in the original, Plainly says so.

Rollout

Simple to adopt. We handle the rest.

Plainly-branded by default

Plainly is introduced as a Plainly-branded experience. Optional co-branding or light customisation is available where appropriate.

We handle support

Plainly handles the product experience and technical support, so your team does not need to field software questions.

No admin system required

Your organisation chooses the approved document set. We handle deployment, updates, and maintenance. No complex dashboard to manage.
