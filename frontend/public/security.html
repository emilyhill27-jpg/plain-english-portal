Security

How we keep your documents safe

No document storage

Documents are processed in real time and are not stored after the response is returned. Plainly does not save documents to any database or cloud storage.

Encrypted in transit

All connections use HTTPS/TLS encryption. Your document content is encrypted between your browser and our server.

No user accounts

We don't require sign-up, login, or personal information. No accounts means no credentials to breach.

No third-party tracking

We do not use analytics scripts or tracking pixels to monitor individual users. No personal browsing data is collected.

AI provider

Plainly uses Anthropic's Claude API to process documents. Anthropic is a US-based AI safety company. Under their API terms, your document content is not stored after processing and is not used for model training.

Data residency

The Plainly application is hosted on Render.com. AI processing calls are made to Anthropic's API (US-based). Document content passes through US infrastructure during processing but is not retained. We are exploring options for AU/NZ-based processing for organisations with data sovereignty requirements.

Accessibility standards

Plainly is designed to meet WCAG 2.2 Level AA and W3C Cognitive Accessibility (COGA) guidelines. This includes colour contrast requirements, keyboard navigation, screen reader support, and neuroinclusive design principles throughout.

Report a concern

If you find a security issue or have a concern about how we handle data, please contact us at hello@tryplainly.co.nz. We take every report seriously.
